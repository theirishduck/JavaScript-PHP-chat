Simple AJAX chat with multi room support

v.0.0.1

Easy to install and easy to extend basic PHP - JavaScript AJAX chat

Installation:

  • Download
  • Unrar the chat.rar
  • Copy the folder to you web app's folder
  • Change the database credentials in config.php
  • Change the s_addr variable in js/config.js to your web app's URL
  • Import db.sql into your database

Usage:

  • To create room log in to your database manager(like: phpMyAdmin) and insert a row into c_rooms table
